# Program Description

This python script is an object oriented program written for the purpose of modeling real world transport payment system like Cowry. It is an object oriented version of the first cowry.py in 'cowry_payment_program' where functions were used. Now, This Cowry class can be used as a base class in multiple cases e.g. Buses, Trains, and Cabs.
